HTML

HTML (HyperText Markup Language) is used to give content to a web page and instructs web browsers on how to structure that content.

HTML Forms

An HTML form is used to collect user input. The user input is most often sent to a server for processing.

The Form Element

The HTML <form> element is used to create an HTML form for user input:

Form attribute:

Attribute Value Description
Action filename.php The action attribute defines the action to be performed when the form is submitted. Usually, the form data is sent to a file on the server when the user clicks on the submit button.
Target _blank The response is displayed in a new window or tab.
_self The response is displayed in the current window.
_parent The response is displayed in the parent frame.
_top The response is displayed in the full body of the window.
framename The response is displayed in a named iframe

The default value is _self which means that the response will open in the current window.

Code Example:

    <form action="/action_page.php" target="_blank">
                .. form content ..
    </form>

Form Tags:

Tag Name Attributes Description
<input> Input value, type, readonly, disabled, size, maxlength, min, max, multiple, placeholder, required, step, autofocus, height, width, list, autocomplete Is an input control that presents users with an interactive control for entering data.
<label> Label for, form is used to provide a usability improvement for mouse users i.e, if a user clicks on the text within the
<select> Select autofocus, disabled, form, multiple, name, required, size is used to create a drop-down list.
<textarea> Textarea autofocus, cols, dirname, disabled, form, maxlength, name, placeholder, readonly, required, rows, wrap defines a multi-line text input control.
<button> Button autofocus, disabled, form, formaction, formenctype, formmethod, formnovalidate, formtarget, name, type, value defines a clickable button.
<fieldset> Fieldset disabled, form, name is used to group related elements in a form.
<legend> Legend supports global and events attributes defines a caption for the <fieldset> element.
<datalist> Datalist supports global and events attributes is used to provide an "autocomplete" feature for <input> elements. Users will see a drop-down list of pre-defined options as they input data.
<output> Output for, form, name is used to represent the result of a calculation (like one performed by a script).
<option> Option disabled, label, selected, value defines an option in a select list.
<optgroup> Optgroup disabled, label is used to group related options in a <select> element (drop-down list).

CSS

CSS (Cascading Style Sheets) describes how HTML elements are to be displayed on screen, paper, or in other media.

CSS Syntax

A CSS rule consists of a selector and a declaration block.

CSS Syntax

The selector points to the HTML element you want to style.

The declaration block contains one or more declarations separated by semicolons.

Each declaration includes a CSS property name and a value, separated by a colon.

Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

Image from www.w3schools.com.

Syntax Example:

    p { 
        color: red;
        text-align: center;
    }

Example Explained:

  • p is a selector in CSS (it points to the HTML element you want to style: <p> ).
  • color is a property, and red is the property value
  • text-align is a property, and center is the property value
